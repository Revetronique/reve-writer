# precheck — 公開前チェック

対象の下書き投稿を `GET /wp-json/wp/v2/posts/{id}?context=edit` で取得し（IDが不明な場合は
`GET /wp-json/wp/v2/posts?status=draft` で最新下書きを一覧提示して確認）、以下を自動確認する。

## チェック項目

1. タイトルにイベント名/場所と年が含まれるか
2. スラッグが英語で設定されているか（英数字・ハイフンのみ。命名規則はCLAUDE.md「スラッグ命名規則」参照）
3. アイキャッチ画像が設定されているか（`featured_media` が 0 でないか）
4. カテゴリ・タグが設定されているか
5. メタディスクリプションが記入されているか（`meta.the_page_meta_description`、120文字以内）
6. 全画像に alt 属性があるか
7. ★マーカーが残っていないか（`[★要加筆`, `[★要確認`, `[★写真`, `[★アフィリエイト` を検索。残っていれば一覧表示）
8. 内部リンク候補の提案（過去記事から関連しそうなもの。`GET /wp-json/wp/v2/posts?search={キーワード}` で検索）
9. 文字数カウント（HTMLタグを除いた本文が2,000〜2,500文字の目安内か。2,500字超はSiteGuardのPOST制限にも抵触し得る）

## 出力形式

チェックリスト形式で表示し、問題があれば修正を提案する:

```
【公開前チェック結果】投稿ID: 1234

  ✅ 1. タイトル: 【イベントレポート】Maker Faire Tokyo 2026に行ってきた（2026年）
  ✅ 2. スラッグ: event-makerfaire-tokyo-2026
  ❌ 3. アイキャッチ: 未設定 → 会場写真（id:5678）の設定を推奨
  ✅ 4. カテゴリ: イベントレポート / タグ: makerfaire, 自作キーボード
  ✅ 5. メタディスクリプション: 98文字
  ⚠️ 6. alt属性: 2枚目の画像に未設定
  ⚠️ 7. ★マーカー残存: 1件
       - 【会場グルメ・購入品】 [★要加筆: 感想]
  💡 8. 内部リンク候補: 「diy-led-matrix-controller」（自作電子工作つながり）
  ✅ 9. 文字数: 2,340字

修正が必要な項目があれば指示をどうぞ（修正は /edit-post のフローで行います）。
```

## 注意

- チェックのみ行い、承認なしに投稿内容を書き換えない
- 修正依頼を受けた場合は `/edit-post` スキルのフローに従う
